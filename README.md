## Usage

Add dependencies to your `pubspec.yaml`

```yaml
dependencies:
    appbar_animated: ^latest
```

Run `flutter pub get` to install.

## How it works

Import library

```dart
import 'package:appbar_animated/appbar_animated.dart';
```
